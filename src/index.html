디바이스 별칭 변경

시리얼:

비우고 저장하면 기본 모델명으로 돌아갑니다 (최대 30자)

무선 디버깅 연결

⭐ 추천: 현재 디바이스(USB든 무선이든)의 포트를 5555로 고정합니다. 이후 VPN/네트워크 변경에도 안정적으로 자동 재연결됩니다.
💡 무선 연결이 안 되어있다면 USB 케이블로 연결 후 시도하세요. (폰 재부팅 후엔 USB 한 번 필요)

또는 수동 페어링: 설정 → 개발자 옵션 → 무선 디버깅 → 페어링 코드로 기기 페어링

디바이스 정보

디바이스를 선택하면 정보가 표시됩니다.

📡 무선 연결 방법

⭐ 권장: 5555 포트 고정 (1회 세팅)

  1. USB로 디바이스 연결 (또는 무선이 이미 연결된 상태)
  2. 상단 무선 연결 → 📌 5555 포트로 고정 클릭
  3. 이후 앱을 켤 때마다 자동 재연결됨 (VPN/네트워크가 바뀌어도 OK)

💡 폰을 재부팅하면 무선 디버깅 포트가 초기화되므로 USB로 한 번 다시 연결해 5555 고정을 해주세요.

대체: 수동 페어링 (USB 없이 첫 연결)

  1. 폰: 설정 → 개발자 옵션 → 무선 디버깅 ON
  2. "페어링 코드로 기기 페어링" 터치 → IP:Port / 6자리 코드 표시
  3. 앱: 무선 연결 모달에서 페어링 IP·코드 입력 → 페어링
  4. 같은 화면 하단 연결용 IP:Port 입력 → 연결

기억된 디바이스 자동 재연결

한 번 연결한 무선 디바이스는 자동으로 기억되어, 앱 시작 시 백그라운드에서 재연결을 시도합니다. 무선 모달의 "기억된 무선 디바이스" 목록에서 직접 연결/제거도 가능.
